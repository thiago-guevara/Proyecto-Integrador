# Sistema de Gestión de Tareas Estudiantiles

## Nombre del proyecto
Sistema de Gestión de Tareas Estudiantiles: organización digital para estudiantes

## Descripción del proyecto
Este proyecto es una aplicación desarrollada en Python que permite gestionar tareas estudiantiles de forma sencilla mediante consola.

El sistema permite al usuario agregar, visualizar, completar y eliminar tareas, facilitando la organización académica.

## Objetivo
Desarrollar un sistema funcional que aplique conceptos básicos de programación como condicionales, ciclos, funciones y listas, además de analizar el impacto de la tecnología en la organización estudiantil.

## Funcionalidades
- Agregar nuevas tareas
- Ver lista de tareas
- Marcar tareas como completadas
- Eliminar tareas

## Tecnologías utilizadas
- Python 3

## Estructura del sistema
El programa funciona mediante un menú interactivo en consola, donde el usuario selecciona opciones para gestionar sus tareas. La información se almacena en una lista con estado de cada tarea.

## Impacto de la tecnología
Este sistema demuestra cómo la tecnología puede mejorar la organización y productividad de los estudiantes, facilitando la gestión del tiempo y tareas académicas.
